Media página web creada por Maria Sofia Ochoa.
Para acceder a las funcionalidades de la página, recomiendo que accedan a /register o que clickeen en el navbar la opción de Registrarse. Una vez hecho eso, inicien sesión. Para editar perfil, usen el botón de respectivo.

Luego, en el navbar aparecen otras opciones que hay que mencionar:
- Acerca de mi -> donde dice un poco de información sobre mi persona
- Blogs -> lista todos los blogs

La sección de Blogs los lista. Sin embargo funciona a medias hasta el día de la fecha (10/05/2022): tiene un blog creado el cual no se puede ver (aunque aparezca, no dice ni el título ni nada). Pero sí se puede añadir una entrada de blog... hasta ahí. Hay problemas con la base de datos y el correcto guardado de las entradas. Por lo tanto, para poder crear una nueva entrada es necesario ingresar a /admin, y desde allí, sería desde un usuario staff, sí se puede crear, ver, detallar y eliminar entradas de blog. 
Todo un tema. 

Ese es en breves palabras el funcionamiento de la página web.
